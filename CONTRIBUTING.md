# Contributing to JOSH

Thank you for your interest in contributing to JOSH (Jurisdictional Objective Standards for Housing). This project exists to give California cities powerful, transparent tools for legally defensible land use decisions.

## How to Contribute

We welcome contributions including bug reports, feature suggestions, documentation improvements, and code.

- **Bug reports and feature requests**: Open an issue on GitHub with a clear description.
- **Code contributions**: Fork the repository, make your changes, and submit a pull request.
- **Documentation**: Improvements to docs, examples, and legal briefs are especially valuable.
